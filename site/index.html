Product

The infrastructure between
raw African signal and better AI output.

Capture

One server-side SDK. Drops into any Node backend. Batches, signs, retries. Captures structured signal from every product surface.

Validate

Typed envelope plus per-event payload schemas. Bad data is rejected at the door with a structured error you can act on.

Ship

Versioned datasets with cryptographic manifests. Sources, time range, schema, row count, content hash — proof travels with the data.

The pipeline

Every signal,
into one lake.

Structured ingestion, immediate queryability, compressed cold storage. Then versioned cuts of the lake ship out as ready-to-train datasets.

Integration

Drop it in.
Start capturing.

Server-side, fail-safe. Capture is on by default, opt-out revocation is one call, and the SDK batches events, signs requests, and never throws into your application.

$ npm install @pandora/sdk
import { init, track } from '@pandora/sdk'; init({ source: 'your-app', endpoint: process.env.PANDORA_URL, keyId: process.env.PANDORA_KEY, secret: process.env.PANDORA_SECRET, }); // batched, signed, validated await track({ userId, type: 'ai.responded', data: { conversationId, text: answer }, });
