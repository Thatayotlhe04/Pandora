Terms

Pandora is data accumulation infrastructure for African product surfaces, partner systems, AI-assisted workflows, and approved public sources. Pandora provides SDK and API access to improve AI output; it is not a translator or a standalone language model.

Model Training Scope

By using a Pandora-enabled platform, users agree that eligible model_training events may be used to build, evaluate, distribute, sell, or commercialize datasets unless the user opts out.

Opt-Out Effect

Opting out stops future model-training collection for that user. Product improvement metadata may continue where needed to operate the service, but eligible raw content is rejected from the externally distributable scope.

Public Sources

Public datasets must preserve attribution, license, source URL, and transformation metadata. Restricted, non-commercial, no-derivatives, or evaluation-only sources are excluded from externally distributed Pandora dataset cuts unless separately approved.
